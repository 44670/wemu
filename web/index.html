How it works

Bring an old game folder back to life.

wemu is a browser game player for classic Windows games. It opens your local files in the current tab, so you can try a game without setting up a virtual machine.

1

Zip your game folder

Keep the game files together, then choose the ZIP in wemu.

2

Pick the program

If there is more than one game program, wemu asks which one to open.

3

Play in the browser

Use your mouse and keyboard in the game window, right from the page.

Why players use it

A simpler way to revisit old PC games.

wemu keeps the first run lightweight: no installer, no account setup, and no upload step before you can see whether a favorite game starts.

Your files stay local

The game ZIP is read by your browser. You do not need to upload it to a server.

No extra setup

Open the player page, choose your ZIP, and start from the browser you already use.

Made for classic games

wemu focuses on older Windows games from the 1990s and early 2000s.

Questions

What to know before you play.

Do I need to install Windows?

No. wemu runs in the browser and opens the game from your local ZIP file.

Does wemu upload my game files?

No. Your ZIP is read locally by the browser so the game can start in the current tab.

Will every old Windows game work?

No browser player can run every game. wemu is built for classic Windows games, and compatibility improves over time.

What kind of file should I choose?

Choose a ZIP that contains the game folder, including the program file and the data files it needs.
